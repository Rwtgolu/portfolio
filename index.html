Project 2

FraudShield is a full-stack fraud detection dashboard built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It enables users to visualize, monitor, and filter financial transactions in real time. The backend integrates intelligent fraud detection logic using statistical analysis and machine learning to flag potentially fraudulent activities. The frontend provides a clean, interactive UI with filters, search, and flagging capabilities. Users can view suspicious transactions, track patterns, and respond promptly to alerts. FraudShield also includes user authentication, role-based access control, and real-time updates through WebSockets, making it a comprehensive and responsive solution for financial fraud monitoring.
